# flute-fingering-p5js

> [!WARNING]
> Used SVG is not unlicened.<br>
> So please consider this is not open sourced. Just utility script for hobby.

Interactively displays flute fingering using [P5.js SVG](https://github.com/zenozeng/p5.js-svg) and [abcjs](https://paulrosen.github.io/abcjs/), based on [YAMAHA music instrument guide](https://www.yamaha.com/ja/musical_instrument_guide/common/images/flute/fingering.pdf).

https://ffunatsu.github.io/flute-fingering-p5js/

![screenshot](docs/screenshot.png)

## Usage

- Click (Tap) upper part: Tone up
- Click (Tap) lower part: Tone down
